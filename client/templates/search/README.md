# Search template

Filtered search results.

## Files

- `Screen.tsx` тАФ the reusable, props-driven component (`SearchResultsScreen`). Copy this into your app.
- `demo.tsx` тАФ a worked example with sample data; this is what the route renders.
- `meta.ts` тАФ registry metadata that drives the Explore grid.

## Use it

```tsx
import { SearchResultsScreen } from "@/client/templates/search/Screen";
```

Route: `/(main)/(demos)/screen-search`
